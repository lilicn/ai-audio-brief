# Parenting Trends Daily Brief Prompt

搜索过去 24 小时的育儿、教育、儿童心理、亲子科技、AI 教育产品和家庭消费趋势新闻。

用中文生成一份适合家长和教育创业者收听的音频简报：

1. 最重要 5-10 条新闻；
2. 每条说明发生了什么、为什么重要、涉及公司/机构/专家；
3. 最新产品、融资、政策、研究或平台变化；
4. 单独总结“家长的钱和注意力正在流向哪里”；
5. 单独总结“教育和亲子科技圈正在发生什么”；
6. 给出对内容创作者、软件工程师、AI 产品经理和教育创业者的启发。

优先引用 Reuters、AP、NPR、EdSurge、The 74、Nature、Pediatrics、官方研究机构和公司公告等高质量来源。
